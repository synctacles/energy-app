## 📖 How to Use

**To give me a task list:**

1. Create a markdown file with tasks (see format below)
2. Upload to this folder or paste in message
3. I'll automatically:
   - Parse all tasks
   - Create GitHub Issues
   - Set priorities & labels
   - Start working with daily updates

**Format Example:**
```markdown
# Tasks - Sprint Name

## 🔴 CRITICAL (Do Immediately)
- [ ] Task description
  - Priority: CRITICAL
  - Effort: 1-2 hours
  - Owner: [role]

## 🟠 HIGH (Next)
- [ ] Task description
  - Priority: HIGH
  - Effort: 3-4 hours

## 🟡 MEDIUM (When time)
- [ ] Task description
  - Priority: MEDIUM
  - Effort: 8+ hours
```
