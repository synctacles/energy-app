# Sessions Directory

This directory contains session summaries for significant Claude Code (CC) and Claude AI (CAI) sessions.

## Naming Convention

```
SESSIE_[CC|CAI]_YYYYMMDD.md
```

- `CC` = Claude Code session
- `CAI` = Claude AI session
- `YYYYMMDD` = Date of session

## When to Create a Session Summary

Create a session summary when:
- Session duration > 1 hour
- Significant changes were made
- Important decisions were taken
- Handoff to other AI or Leo is needed

## Template

See SKILL_00 Section Q for the full template.

## Archive

Sessions older than 30 days should be moved to `archive/` subdirectory.
